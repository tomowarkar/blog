用途

bbenoist.shell
  • エディタから直接shellコマンドを実行できる
  • ほぼ使っていない
christian-kohler.path-intellisense
  • ファイル名を自動補完
  • 賢くて良き
dbaeumer.vscode-eslint
  • コードフォーマッター
  • よくわからん
esbenp.prettier-vscode
  • コードフォーマッター
  • Prettier 入門 ～ESLintとの違いを理解して併用する～
eg2.vscode-npm-script
  • npm のhogehoge
  • よくわからん
christian-kohler.npm-intellisense
  • npmモジュールを自動補完
dariofuzinato.vue-peek
  • vue のhogehoge
  • vueをあまり書いてないからよくわからん
jcbuisson.vue
  • vue のSyntax Highlight
  • まあいるよね
mubaidr.vuejs-extension-pack
  • vue 関連
octref.vetur
  • 同上
formulahendry.auto-complete-tag
  • 以下二つのまとめ役
formulahendry.auto-close-tag
  • html などの close tag 自動補完
  • 必須
formulahendry.auto-rename-tag
  • 便利
  • 時々うざい
formulahendry.code-runner
  • VScode上でのショートカット操作で外部のshell scriptを実行できる
  • 入れたてホヤホヤ未使用様
KnisterPeter.vscode-github
  • github関連
  • どこからどこまで何をしているのかわからない
ms-azuretools.vscode-docker
  • docker for vscode
  • あまりdocker触ってないなぁ
MS-CEINTL.vscode-language-pack-ja
  • VSCodeの日本語対応
  • 必須
ms-mssql.mssql
  • sql関連
ms-python.python
  • python関連
  • 必須
tht13.python
  • python関連
  • よくわからん
donjayamanne.jupyter
  • jupyter notebook　のhogehoge
  • あると便利
  • なくても困らん
xaver.clang-format
  • c関連
ms-vscode.cpptools
  • c/c++関連
ms-vscode.Go
  • go 関連
  • 必須
ms-vscode.vscode-typescript-tslint-plugin
  • 使ってないなぁ
robertoachar.vscode-essentials-snippets
  • 見た感じ便利そう
  • なお使ったことはない
tombonnike.vscode-status-bar-format-toggle
  • なんだ… これは…
tomoki1207.pdf
  • VSCodeでPDFが見れる
  • benri
VisualStudioExptTeam.vscodeintellicode
  • ものゴッツ便利
xabikos.JavaScriptSnippets
  • JS
  • こんなん知らんかった
  • 今度使いたい
yzhang.markdown-all-in-one
  • markdown関連
  • 必須
